# Welcome

**BookOrbit** for Android is a native Kotlin/Jetpack Compose client for a self-hosted
[BookOrbit](https://github.com/bookorbit/bookorbit) library server. It's not a hosted service —
you connect it to a server that you (or someone you trust) runs and controls.

The app supports:

- Sign-in via username/password or OIDC
- Library browsing, search, smart scopes, and collections
- Author/series pages and book details
- An EPUB reader (foliate.js) with theming and reading progress sync
- An audiobook player (Media3/ExoPlayer) with background playback
- Downloads for offline reading and listening

Use the sidebar to get set up:

1. [Install the app](./installation.md)
2. [Connect to your server](./connecting-to-a-server.md)
3. [Use the app](./using-the-app.md)
